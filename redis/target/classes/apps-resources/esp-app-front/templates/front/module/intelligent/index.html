      TCS国际贸易全链通综合服务平台

      • 报关
        报关

        报关申报系统可按需要开设多个申报端口，网上直接报关，打破 QP 系统单机操作的限制。三大功能“一次申报，备案清单，集报清单”与企业各种管理系统（如 ERP 等）的数据无缝对接，报关信息快速导入，自动生成报关资料如合同、发票、装箱单、舱单等，数据一次录入，多模块共享。全过程海关申报状态实时跟踪，方便企业相关人员及时跟进通关状态，提高通关时效。有效帮助企业达成“提速、提质、增效、减负”的效果。

      • 商检
        商检

        TCS全链通平台，事先维护商品数据库，按商检要求维护申报要素，企业ERP导出EXCEL格式，直接在TCS系统导入，即可以快捷生成商检要求的报检单，原产地证书等申报信息，一键导出申报至国检等平台，实现一次录入多方申报、免人工干预，减少人为差错，有效控制检务风险。

      • 公路舱单
        公路舱单

        开设客户端舱单信息网上录入窗口，实现各自录入、统一申报、进度状态实时同步查询。 舱单信息快速导入，无需手工逐项录入。异常申报状态自动置前提示，避免漏单或处理不及时。车辆捆绑状态实时自动短信通知。定期票数汇总统计查询。对舱单确报录入信息进行全面准确的逻辑校验。系统对舱单及报关单设置相应的查询条件，客户可根据自身需要随时搜索导出查询结果，方便企业统计分析。

      • 海运舱单
        海运舱单

        开设客户端舱单信息网上录入窗口，实现各自录入、统一申报、进度状态实时同步查询。 舱单信息快速导入，无需手工逐项录入。异常申报状态自动置前提示，避免漏单或处理不及时。定期票数汇总统计查询。对舱单录入信息进行全面准确的逻辑校验。可导出舱单海关回执和申报明细。

      • 进出口关务管理
        进出口关务管理

        TCS全链通进出口关务管理系统是为供应链企业、报关报检服务公司提供的一套客户管理、订单管理、单据批量处理、状态查询、报价结算、数据分析、财务分析等功能的信息化解决方案，为客户提供一套规范化的内部管理方案，减低企业成本。

      • 金关二期加贸管理
        金关二期加贸管理

        按照金关二期的总体建设要求，建设完整的加工贸易手/账册体系，建立符合海关、企业需求和海关监管要求的加工电子手册/账册，优化“备案、通关、核销”三段式管理模式，实行“料号级管理”的新型监管模式，实现“精准化”核销核算，实现预录入及数据传输功能。构建相对规范又局部个性化的管理模式，有效解决海关保税加工的监管难题，探寻把关与服务新的平衡点。
